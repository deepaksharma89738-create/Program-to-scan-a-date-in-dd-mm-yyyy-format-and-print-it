# Program-to-scan-a-date-in-dd-mm-yyyy-format-and-print-it
Program reads a date from the user in dd/mm/yyyy format and prints it. In C, the date values such as day, month, and year are taken as input using scanf() with the format %d/%d/%d. After reading the values, the program displays the entered date using printf(), ensuring the same format dd/mm/yyyy.
